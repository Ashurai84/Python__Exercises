# Jupyter Notebook Code Extractor

This project provides a script to extract code cells from a Jupyter notebook (`.ipynb` file) and save them as individual `.py` files.

## Prerequisites

Make sure you have the following installed:
- Python 3.x
- Jupyter Notebook
- `nbformat` package
